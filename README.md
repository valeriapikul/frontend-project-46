## Description
Compares 2 files for differences. Supports formats:
- json
- yml/yaml
The comparison result can be formatted as:
- JSON
- plain
- stylish

## Minimum system requirements
node.js

## Installation
make install

## Launch
gendiff -h - show help
gendiff path to file #1 path to file #2
