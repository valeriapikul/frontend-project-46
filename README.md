## Описание
Сравнивает 2 файла на наличие различий. Поддерживает форматы:
- json
- yml/yaml
Результат сравнения можно отформатировать в:
- JSON
- plain
- stylish

## Минимальные системные требования
node.js

## Установка
make install

## Запуск
gendiff -h - показать справку
gendiff путь до файла №1 путь до файла №2
